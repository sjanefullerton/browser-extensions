In process of being approved as a chrome browser extension. For now, try it locally:

1. Unzip browser-extension.zip
2. Go to chrome://extensions/
3. Turn Developer Mode ON
4. Click “Load unpacked”
5. Select the browser-extension-merged folder
6. Click the extension, enter the ID(s) you want CSVs from, and select sections (Analytics, Monetization, Custom Events)
